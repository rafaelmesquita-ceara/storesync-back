# StoreSync — Visão Geral

Sistema de gestão para lojas de pequeno porte. Backend: ASP.NET Core 9 + PostgreSQL + Dapper. Frontend: Avalonia UI 11 + MVVM. PKs: `Guid`. Datas: `BrazilDateTime.Now` (America/Sao_Paulo). Senhas: BCrypt. TDD obrigatório.
